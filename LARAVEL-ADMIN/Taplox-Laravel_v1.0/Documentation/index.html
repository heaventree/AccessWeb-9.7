Overview

Taplox is a modern, developer-centric, and highly customizable HTML dashboard template. Built on Bootstrap 5, PHP v8, Laravel 11, SCSS, and jQuery, it features a fully responsive layout and an intuitive interface. With a vast library of components, widgets, and UI elements, Taplox accelerates project development, saving valuable time and resources. Perfect for SaaS interfaces, admin dashboards, or any other web application, Taplox streamlines your workflow.

Your feedback is invaluable to us. Please don’t hesitate to reach out with any comments, questions, or suggestions.

Prerequisites

To get started with Taplox, please follow the steps below to install and set up all the necessary prerequisites:

  • PHP >= 8.2
  • BCMath PHP Extension
  • Ctype PHP Extension
  • Fileinfo PHP extension
  • JSON PHP Extension
  • Mbstring PHP Extension
  • OpenSSL PHP Extension
  • PDO PHP Extension
  • Tokenizer PHP Extension
  • XML PHP Extension

  • Composer

    Make sure you have the Composer installed & running on your computer. If you already have installed Composer on your computer, you can skip this step.

    Download Composer

  • Node.js

    In order to use build tools you will need to download and install Node.js. If you do not have Node.js installed already, you can get it by downloading the package installer from the official website. Please download the stable version of Node.js (LTS).

    Download Node.js
Installation

To setup, follow the below mentioned steps:

You can run the following commands to run project locally or build for production use:

You can run this app using one of these package managers: Bun, Yarn or NPM

Bun
Command Description
bun i This would install all the required dependencies in the node_modules folder.
bun dev Runs the project locally, starts the development server.
bun run build It bundles with production mode. Your app is now ready to be deployed.
Yarn
Command Description
yarn This would install all the required dependencies in the node_modules folder.
yarn dev Runs the project locally, starts the development server.
yarn build It bundles with production mode. Your app is now ready to be deployed.
NPM
Command Description
npm i This would install all the required dependencies in the node_modules folder.
npm run dev Runs the project locally, starts the development server.
npm run build It bundles with production mode. Your app is now ready to be deployed.

To spin up the Laravel server, follow the below mentioned steps in a separate terminal:

Command Description
composer i This would install all the required dependencies in the vender folder.
cp .env.example .env This will copy .env.example as .env file.
php artisan key:generate This will generate a unique key and write it in your .env file
php artisan migrate This will executes all pending migrations to database.
php artisan serve Runs the project locally, starts the development server on http://127.0.0.1:8000
